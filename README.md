# AI Masterclass Code (Beginner → Senior)

Practical, week-by-week code templates and exercises.

## Structure

- `00-start-here/` — orientation + how to run the exercises
- `01-python/` — Python fundamentals through reliability basics

## Quickstart

Create and activate a virtual environment, then run the week’s `main.py`.

### Windows (PowerShell)

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python --version
```

### macOS / Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
python --version
```

## Weeks 1–8

- Week 01 — Start here (project orientation)
- Week 02 — Setup (venv, env vars, running scripts)
- Week 03 — Python essentials (text utilities + dataclasses)
- Week 04 — Files (read/process/write)
- Week 05 — APIs & retries (timeouts, backoff)
- Week 06 — Git & GitHub (workflow cheat sheet)
- Week 07 — Capstone summarizer (CLI + provider abstraction)
- Week 08 — Reliability basics (validation, logging, run reports)
