# Week 03 — Python Essentials

Goal: practice writing small, testable utility functions and using `dataclasses` to hold configuration.

## What you’ll build

- `text_utils.py` with:
  - `clean_text()` to normalize whitespace and optionally lowercase
  - `split_sentences()` a simple splitter on `.`, `!`, `?`
- `main.py` that:
  - defines a `Config` dataclass
  - cleans a sample text
  - split + filters sentences
  - prints results

## Run

```bash
python main.py
```
